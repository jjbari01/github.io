System Analysis

Fundamentals of object-oriented analysis and design. Topics include requirements determination, feasibility analysis, modeling with Unified Modeling Language (UML) and data dictionary construction, data modeling and normalization, user interface requirements specification, and information security procedures. As part of a team, I created a new system proposal for the non-profit Land O'Lakes Area Artisans (LOLA) group in northern Wisconsin. We analyzed the current system and created the documentation to show how they could update their system to continue their mission in growing their art out reach programs for rural Wisconsin.
