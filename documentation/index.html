Overview


Celsius is is a magnificent HTML5 responsive template built with portfolio and photography in mind. Celsius comes bundled with astounding features like page preloader, video background, full page slider and much more.

The underlying UIkit gives the template an immeasurable weight of flexibility and agility, fast loading site using built in compression of files, data URIs and gzip compression. Celsius is also available for Joomla!

Folder Structure


Celsius template comprises of LESS, CSS and Javascript that power the robust underlying UIKit framework. In this section, we are going to cover the css, LESS and js to help you understand the template’s structure.

  1. CSS Folder

    This folder contains css files which render the front-end look and feel of the site. It contains four files:

    File Description
    bootstrap.css this file contains the css for bootstrap compatibility.
    Add this file if you need to integrate the template with bootstrap
    custom.css use this file to add your own custom.css.
    ie9.css this file fixes issues related with Internet Explorer 9 browser.
    theme.css this file contains the css for the template.
    All other styles e.g theme-azure.css, are derived from this file
    theme-rtl.css this file contains the css for the rtl version of the template.

    IMPORTANT
    When doing your customization, it's recommended not to edit theme.css directly. Place all the custom css in css/custom.css

  2. Fonts Folder

    This folder contains icon fonts files.

  3. Images Folder

    This folder contains the image files that have been used in the template.

  4. js Folder

    This folder contains the javascript files used in the template.

  5. Less Folder

    This folder contains the less files which in turn, generate the css output files that are located in the css folder. The less files are:

    File Description
    theme.less is the key file that combines uikit less files (located in the uikit sub folder)
    custom-theme.less you can use this file to add your own less code
    style.less tthis file defines the unique less code for each style

  6. Licensing Folder

    This folder contains the licence files which govern the use of this template.

    7. Vendor Folder

    This folder contains the UIKit framework files.


    Module Positions


    This template comes with a sophisticated layout system to create any kind of sidebar or widget layout. You can easily manage the sidebar's positions and widths in the template administration.

    Widgets can have different styles and be placed in any position offered by this template. Each position has its own layout. You can align widgets side-by-side, stack them or choose your own grid layout. Show or hide widgets on mobile devices, while the grid adapts perfectly to your layout.
