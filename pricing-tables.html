Pricing tables with focus

You can set a pricing table to have focus by using the focus on the same div as tm-price-column class. You can further shift the focus pricing table upwards to create symmetry by using the tm-shift-top class. Notice also how the focused table has more items to make it larger and thus fitting with the shift.
