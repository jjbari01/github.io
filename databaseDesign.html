Database Design
CIS 310: Database Design

This database course provides a solid and practical foundation for the design and implementation of database systems. Emphasis on relational database models with significant coverage of basic relational database concepts, normalization, E-R modeling, SQL experience and distributed databases. Extensive experience within the SQL server.
